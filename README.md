# webdev-course-kauflink

Repositorio de código fuente del curso **Fundamentos de Desarrollo Web para Principiantes (HTML y CSS)**, elaborado como parte del trabajo complementario del curso 1ASI0730 Aplicaciones Web — Universidad Peruana de Ciencias Aplicadas, período 202610.

**Plan del curso completo**: [course-plan.md](./course-plan.md)

---

## Configuración

No se requiere instalar ninguna herramienta. Todo el código se escribe y se ejecuta en línea desde el navegador con [CodePen](https://codepen.io), gratis y sin necesidad de registrarse. Cada lección incluye un enlace directo a su proyecto precargado.

> **¿Por qué CodePen?** Tiene paneles separados para HTML y CSS y muestra el resultado **en vivo** mientras escribes, sin botón de "ejecutar" ni configuración. Es el editor en línea más sencillo para empezar con HTML y CSS. Alternativas igual de válidas: [JSFiddle](https://jsfiddle.net) y [Replit](https://replit.com).

Para abrir los archivos localmente (opcional):
1. Descarga el archivo `.html` de la carpeta correspondiente.
2. Haz doble clic sobre él para abrirlo en cualquier navegador (Chrome, Firefox, Safari, Edge).
3. Edítalo con cualquier editor de texto y recarga la página para ver los cambios.

## Estructura del repositorio

```
webdev-course-kauflink/
|
|-- README.md                  <- Este archivo
|-- course-plan.md             <- Plan completo del curso con lecciones y enlaces
|
|-- starter-files/             <- Archivos de práctica para los estudiantes
|   |-- Lesson1_PrimeraPagina.html
|   |-- Lesson2_EstructuraHTML.html
|   |-- Lesson3_ElementosComunes.html
|   |-- Lesson4_IntroCSS.html
|   |-- Lesson5_PaginaPerfil.html
|   +-- Lesson6_BuenasPracticas.html
|
|-- completed-examples/        <- Soluciones completas de referencia
|   |-- Lesson1_PrimeraPagina_done.html
|   |-- Lesson2_EstructuraHTML_done.html
|   |-- Lesson3_ElementosComunes_done.html
|   |-- Lesson4_IntroCSS_done.html
|   |-- Lesson5_PaginaPerfil_done.html
|   +-- Lesson6_BuenasPracticas_done.html
|
+-- scripts/                   <- Notas de cada lección para el instructor
    +-- lesson-notes.md
```

### starter-files/
Archivos HTML con el esqueleto de la página. Los estudiantes los abren en el editor en línea y completan las secciones marcadas con `<!-- TODO -->`. No incluyen la solución a propósito.

### completed-examples/
Versiones completamente funcionales de cada ejercicio. Sirven como referencia para el estudiante después de intentar resolverlo por su cuenta.

### scripts/
Notas de apoyo para el instructor con los puntos clave de cada lección y sugerencias de enseñanza.
